# RoboSkill - National Robotics Competition Website

A beautifully designed, fully responsive, and **dual-language (Persian & English)** website for the **RoboSkill Robotics Competition**, a government-run national event in Iran. Built in **2018** using **HTML5**, **CSS3**, and **jQuery**, featuring a custom UI tailored for robotics competitions.

## 🤖 Overview
This is a large-scale, multi-page static website created for a national robotics competition. It includes support for both right-to-left (RTL) and left-to-right (LTR) layouts to accommodate Persian and English languages seamlessly.

The design was crafted with attention to detail and optimized for all screen sizes — from mobile to desktop.

## 🌐 Features
- Dual-language support: **Persian (fa-*.html)** and **English**
- Fully responsive layout
- RTL (Right-to-Left) and LTR (Left-to-Right) compatibility
- Custom graphics and animations
- Integrated JavaScript libraries for carousel, countdown, and more
- Clean HTML structure and semantic markup

## 💻 Technologies Used
- **HTML5** – Semantic structure and accessibility
- **CSS3** – Responsive layout, transitions, Flexbox/Grid

- **Owl Carousel** – Image sliders
- **FlipClock.js** – Countdown timer
- **html5shiv.js** – HTML5 support for older browsers
- **respond.js** – Media query support for IE
